Object Detection for the Duckiebot

Yolov5 object detection model and Braitenberg controller implemented on the Duckiebot platform.

ROS Python

Final year undergraduate project to investigate the Duckiebot platform for Machine Learning by implementing an object detection model. A pretrained YOLOv5 model, fine-tuned with a publicly available Duckietown dataset was used to detect Duckeitown specific objects in Python. To enable autonomous object avoidance, a simple Braitenberg controller was used. The final solution could detect objects (duckies) in the scene and compute wheel commands to avoid the objects for relatively simple scenarios. The project formed a foundational model that future students can improve and build on for more sophisticated behaviours.
